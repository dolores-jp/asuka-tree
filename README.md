# AsukaTree

<p align="center">
  <img src="AsukaTree/icons/AsukaTree_256.png" alt="AsukaTree icon" width="128">
</p>

AsukaTree は、[AsukaTools](https://web.archive.org/web/20241015162142/http://www.lapcie.com/index.php?page=AsukaTools) の HTTP 機能を利用して、所持アイテムのデータを JSON で取得した後、見やすく整形してウィンドウに出力する windows のデスクトップアプリです。

利用には HTTP 機能が有効化されている version 2.3 以降の AsukaTools が必要です。

## 動作環境

-  Windows 11（64bit)
	- 多分 Windows 10 でも動くと思うけど未確認

## 起動方法

1. 配布された ZIP を展開
2. `AsukaTree.exe` を実行
